# SmartFeed
Gerador de Fórmulas de Ração Animal

## Sobre o Projeto
O SmartFeed é um software desenvolvido em Java com o objetivo de auxiliar na criação de fórmulas de ração animal otimizadas, considerando fatores como disponibilidade e preço dos ingredientes, valores nutricionais e o tipo de animal. O sistema busca apoiar produtores rurais, zootécnicos e fábricas de ração na elaboração de misturas equilibradas e economicamente viáveis, utilizando princípios da Programação Orientada a Objetos (POO).

O programa permite o cadastro de ingredientes (por exemplo: milho, farelo de soja, óleo de peixe, etc.), incluindo informações nutricionais (proteína, energia, fibra, etc.) e preço de mercado. Também possibilita a gestão de tipos de animais (bovinos, aves, suínos, entre outros), com suas respectivas exigências nutricionais. A partir desses dados, o sistema gera uma fórmula de ração que atende às necessidades nutricionais do animal ao menor custo possível.

A arquitetura do sistema segue os princípios da POO, representando entidades do mundo real através de classes bem definidas. As relações entre essas classes (composição, herança e polimorfismo) tornam o sistema modular, reutilizável e fácil de expandir — por exemplo, permitindo adicionar novos tipos de animais ou métodos de cálculo de fórmulas.
